PLaNet - Phenotypic Linkage Network Viewing Tool

Welcome to PLaNet, a web interface for the display, manipulation and construction of Phenotypic Linkage Networks.

Example Dataset

PLaNet contains a number of pre-built Phenotypic Linkage Networks along with associated statistical analysis results. Datasets that are included in PLaNet:
  • General PLN:
    • Coexpression Gene-Set Enrichment (coexpr_gse)
    • Semantic Gene Ontology Analysis for Biological Process (sem_goabp)
    • Set 9 Genes (s9_genes)
    • Parkinson's Disease Differential Expression 1 (pd_de_1)
    Gene Visualisation
    PLaNet allows for the display of specific subsets of genes within the Phenotypic Linkage Network, either uploaded manually by the user or via the selection of a pre-curated example gene set.
  • User Upload and Search:
    • Selecting this button will enable the user to manually search for genes (up to 30, comma seperated), using the text box that appears, within the currently selected dataset.
    • Select Gene Set:
    • PLaNet contains a number of curated example gene lists (PD1, PD2, AD1, AD2) that can be searched for when you select the 'Select Gene Set' option.
    • Selecting 'None' will display the whole, unsubsetted dataset that you've selected.
  • Select Number of Node Neighbours:
    • When visualising a subset of genes within the Phenotypic Linkage Network, PLaNet also also allows you to select the degree of seperation between the visualised genes and those with a direct relationship (the 'neighbour' of that node'.
    • Operating the slider will allow you to select the degree of 'Node Neighbours' to the visualised set of genes (selecting 0 will just show the visualised genes with no neighbours).
    Phenotypic Linkage Network
    This is the tool for viewing the Phenotypic Linkage Network datasets, using the visNetwork network plotting tool (https://github.com/datastorm-open/visNetwork).
  • Nodes:
    • 'Nodes' are the singular entities that are plotted to the network. Depending on the type of node plotted, the colour, shape and annotations for each of these points may change.
  • Edges:
    • The 'Edges' of the network, more commonly referred to as the 'Relationships' between the nodes, are lines that show an interaction, association or weighting between these two points in the dataset. The strength of these edges determines the weight of the lines, and the dataset they have come from determines the colour: These can be viewed by hovering over the edges to view the annotations.
  • Number of Points to Display:
    • As part of the viewer is a slider that allows for the selection of how many points you wish to display in the network. Please note, a greater number of nodes selected to display will mean the network will take longer to load and display.
    Phenotypic Linkage Network Selection
    This is where you can select which of the Phenotypic Linkage Networks you want to display in the dataset. If none are selected, nothing will be plotted. If all are selected, then the whole General PLN dataset will be available to the Phenotypic Linkage Network viewer.
